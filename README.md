# Laravel - MyBookStore

MyBookstore Is A Basic E Commerce Site That Allow Admin To Manage Products And Visiters And Buy Bookes And Payment Done By Strip

## Features

- Product Management
- Admin Controll
- Strip Api
- Passwords Encryption
- Laravel Librarys

- [html,css,Bootstrap](#) - Front End
- [Php, Laravel](#) - BackEnd
- [Strip](#) - Payment GetWay
- [Mysql](#) - Data Storage
